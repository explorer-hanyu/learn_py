# Python3 数字(Number)

## 一、数字数据类型概述

Python 数字数据类型用于存储数值。

### 核心特性

- **不可变类型**：数字数据类型是不允许改变的，改变数字的值将**重新分配内存空间**
- **对象创建**：变量赋值时 Number 对象被创建
  ```python
  var1 = 1
  var2 = 10
  ```

### 删除数字对象

使用 `del` 语句删除数字对象的引用：

```python
del var1          # 删除单个对象
del var_a, var_b  # 删除多个对象
```


## 二、Python3 支持的数值类型

Python3 支持三种不同的数值类型：

| 类型 | 说明 | 示例 |
|------|------|------|
| **int（整型）** | 正或负整数，不带小数点。Python3 整型**没有大小限制**，可当作 Long 类型使用。Python3 没有 Python2 的 Long 类型。**bool 是 int 的子类型** | `10`, `100`, `-786`, `0x69` |
| **float（浮点型）** | 由整数部分与小数部分组成，也支持**科学计数法**表示（如 `2.5e2 = 250`） | `0.0`, `15.20`, `-21.9`, `32.3e+18` |
| **complex（复数）** | 由实数部分和虚数部分构成，用 `a + bj` 或 `complex(a, b)` 表示。实部 a 和虚部 b 都是**浮点型** | `3.14j`, `45.j`, `9.322e-36j` |

### 整数的进制表示

Python 支持使用十六进制和八进制直接表示整数：

```python
>>> number = 0xA0F    # 十六进制
>>> number
2575

>>> number = 0o37     # 八进制
>>> number
31
```

### 数值类型字面量示例

| int | float | complex |
|-----|-------|---------|
| 10 | 0.0 | 3.14j |
| 100 | 15.20 | 45.j |
| -786 | -21.9 | 9.322e-36j |
| 0x69 | 70.2E-12 | 4.53e-7j |


## 三、数字类型转换

Python 中数据类型的转换只需**将数据类型名作为函数名**即可：

| 函数 | 描述 |
|------|------|
| `int(x)` | 将 x 转换为一个整数 |
| `float(x)` | 将 x 转换到一个浮点数 |
| `complex(x)` | 将 x 转换到一个复数，实数部分为 x，虚数部分为 0 |
| `complex(x, y)` | 将 x 和 y 转换到一个复数，实数部分为 x，虚数部分为 y |

### 示例

```python
>>> a = 1.0
>>> int(a)
1
```


## 四、数字运算

Python 解释器可以作为一个**简单的计算器**使用。

### 基本算术运算符

| 运算符 | 说明 | 示例 |
|--------|------|------|
| `+` | 加法 | `2 + 2` → `4` |
| `-` | 减法 | `50 - 5*6` → `20` |
| `*` | 乘法 | `5 * 3 + 2` → `17` |
| `/` | 除法（**总是返回浮点数**） | `8 / 5` → `1.6` |
| `//` | 整除（向下取整） | `17 // 3` → `5` |
| `%` | 取余 | `17 % 3` → `2` |
| `**` | 幂运算 | `5 ** 2` → `25` |

### ⚠️ 注意点

- **除法 `/`**：总是返回一个浮点数
- **整除 `//`**：返回**向下取整**后的结果，得到的类型与分母分子的数据类型有关
  ```python
  >>> 7//2     # 3
  >>> 7.0//2   # 3.0
  >>> 7//2.0   # 3.0
  ```
- **混合运算**：不同类型的数混合运算时，会将整数转换为浮点数
  ```python
  >>> 3 * 3.75 / 1.5   # 7.5
  ```

### 变量赋值

等号 `=` 用于给变量赋值，赋值后解释器不会显示任何结果：

```python
>>> width = 20
>>> height = 5 * 9
>>> width * height
900
```

### 变量必须先定义后使用

```python
>>> n   # 尝试访问未定义的变量
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
NameError: name 'n' is not defined
```


## 五、交互模式特殊变量 `_`

在交互模式中，**最后被输出的表达式结果**会自动赋值给变量 `_`：

```python
>>> tax = 12.5 / 100
>>> price = 100.50
>>> price * tax
12.5625
>>> price + _
113.0625
>>> round(_, 2)
113.06
```

> **注意**：`_` 变量应被视为**只读变量**。


## 六、数学函数（部分）

| 函数 | 描述 |
|------|------|
| `abs(x)` | 返回数字的绝对值，如 `abs(-10)` 返回 `10` |
| `math.ceil(x)` | 返回数字的上入整数，如 `math.ceil(4.1)` 返回 `5` |
| `cmp(x, y)` | Python 3 已废弃，使用 `(x>y)-(x<y)` 代替 |
