# Python3 推导式

## 一、推导式概述

推导式（Comprehension）是 Python 中一种**独特且高效的数据处理方式**，可以从一个数据序列构建另一个新的数据序列。

Python 支持四种推导式：

| 推导式类型 | 语法符号 | 返回类型 |
|-----------|---------|---------|
| 列表推导式 | `[]` | 列表（list） |
| 字典推导式 | `{}` | 字典（dict） |
| 集合推导式 | `{}` | 集合（set） |
| 元组推导式（生成器表达式） | `()` | 生成器对象（generator） |


## 二、列表推导式

### 2.1 基本语法

```python
[表达式 for 变量 in 可迭代对象]
```

或带过滤条件：

```python
[表达式 for 变量 in 可迭代对象 if 条件]
```

- `表达式`：列表生成元素表达式，可以是有返回值的函数
- `for 变量 in 可迭代对象`：迭代输入序列
- `if 条件`：可选，过滤不符合条件的值

### 2.2 示例

**过滤长度大于 3 的字符串并转为大写：**

```python
>>> names = ['Bob','Tom','alice','Jerry','Wendy','Smith']
>>> new_names = [name.upper() for name in names if len(name) > 3]
>>> print(new_names)
['ALICE', 'JERRY', 'WENDY', 'SMITH']
```

**计算 30 以内能被 3 整除的数：**

```python
>>> multiples = [i for i in range(30) if i % 3 == 0]
>>> print(multiples)
[0, 3, 6, 9, 12, 15, 18, 21, 24, 27]
```


## 三、字典推导式

### 3.1 基本语法

```python
{key_expr: value_expr for value in collection}
```

或带过滤条件：

```python
{key_expr: value_expr for value in collection if condition}
```

### 3.2 示例

**以字符串为键、长度为值创建字典：**

```python
>>> listdemo = ['Google','Runoob', 'Taobao']
>>> newdict = {key: len(key) for key in listdemo}
>>> newdict
{'Google': 6, 'Runoob': 6, 'Taobao': 6}
```

**以数字为键、平方为值创建字典：**

```python
>>> dic = {x: x**2 for x in (2, 4, 6)}
>>> dic
{2: 4, 4: 16, 6: 36}
```


## 四、集合推导式

### 4.1 基本语法

```python
{expression for item in Sequence}
```

或带过滤条件：

```python
{expression for item in Sequence if conditional}
```

### 4.2 示例

**计算数字的平方：**

```python
>>> setnew = {i**2 for i in (1, 2, 3)}
>>> setnew
{1, 4, 9}
```

**过滤掉特定字母：**

```python
>>> a = {x for x in 'abracadabra' if x not in 'abc'}
>>> a
{'d', 'r'}
```


## 五、元组推导式（生成器表达式）

### 5.1 基本语法

```python
(expression for item in Sequence)
```

或带过滤条件：

```python
(expression for item in Sequence if conditional)
```

### 5.2 重要特性

元组推导式与列表推导式的用法完全相同，区别在于：

| 对比项 | 列表推导式 | 元组推导式 |
|--------|-----------|-----------|
| 符号 | `[]` 中括号 | `()` 圆括号 |
| 返回结果 | 列表 | **生成器对象**（不是元组） |

如果需要得到元组，需要用 `tuple()` 转换。

### 5.3 示例

```python
>>> a = (x for x in range(1, 10))
>>> a
<generator object <genexpr> at 0x7faf6ee20a50>   # 返回的是生成器对象
>>> tuple(a)                                      # 转换为元组
(1, 2, 3, 4, 5, 6, 7, 8, 9)
```


## 六、四种推导式对比总结

| 推导式 | 语法符号 | 返回类型 | 特点 |
|--------|---------|---------|------|
| 列表推导式 | `[expr for x in iter]` | `list` | 最常用，一次性生成所有元素 |
| 字典推导式 | `{key_expr: val_expr for x in iter}` | `dict` | 生成键值对 |
| 集合推导式 | `{expr for x in iter}` | `set` | 自动去重 |
| 元组推导式 | `(expr for x in iter)` | `generator` | 惰性求值，节省内存 |

> **使用建议**：推导式虽然简洁强大，但需要注意**可读性**，尽量保持表达式简洁，以免影响代码的可读性和可维护性。
