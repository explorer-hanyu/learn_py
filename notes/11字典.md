# Python3 字典(Dictionary)笔记

## 一、字典概述

字典是另一种**可变容器模型**，且可存储**任意类型对象**。

字典的每个键值对（key=>value）用**冒号 `:`** 分割，每个对之间用**逗号 `,`** 分割，整个字典包括在**花括号 `{}`** 中。

```python
d = {key1: value1, key2: value2, key3: value3}
```

### 字典的核心特性

| 特性 | 说明 |
|------|------|
| **可变性** | 字典是**可变的**，可以增删改元素 |
| **键唯一性** | 键**必须唯一**，值则不必 |
| **键的不可变性** | 键必须是**不可变类型**（如字符串、数字、元组），列表等可变类型不能作为键 |
| **值的任意性** | 值可以是**任意数据类型** |
| **有序性** | Python 3.7+ 字典**保持插入顺序** |


## 二、创建字典

### 2.1 使用花括号 `{}` 创建

```python
# 直接创建
tinydict = {'name': 'runoob', 'likes': 123, 'url': 'www.runoob.com'}

# 空字典
emptyDict = {}
```

### 2.2 使用 `dict()` 内建函数创建

```python
emptyDict = dict()
```

### 2.3 注意事项

- `dict` 是 Python 的关键字和内置函数，**变量名不建议命名为 `dict`**
- 键可以是数字、字符串或元组，但不能是列表


## 三、访问字典里的值

把相应的键放入方括号 `[]` 中即可：

```python
tinydict = {'Name': 'Runoob', 'Age': 7, 'Class': 'First'}
print(tinydict['Name'])   # Runoob
print(tinydict['Age'])    # 7
```

### ⚠️ 访问不存在的键会报错

```python
print(tinydict['Alice'])   # KeyError: 'Alice'
```


## 四、修改字典

向字典添加新内容的方法是**增加新的键/值对**，修改或删除已有键/值对：

```python
tinydict = {'Name': 'Runoob', 'Age': 7, 'Class': 'First'}

# 修改已有键的值
tinydict['Age'] = 8

# 添加新的键值对
tinydict['School'] = "菜鸟教程"

print(tinydict['Age'])     # 8
print(tinydict['School'])  # 菜鸟教程
```


## 五、删除字典元素

能删单一的元素也能清空字典：

| 操作 | 语法 | 说明 |
|------|------|------|
| 删除单个键值对 | `del dict['key']` | 删除指定键及其对应的值 |
| 清空字典 | `dict.clear()` | 删除字典内所有元素，字典变为空字典 |
| 删除整个字典 | `del dict` | 删除字典本身，之后无法再访问该字典 |

```python
tinydict = {'Name': 'Runoob', 'Age': 7, 'Class': 'First'}

del tinydict['Name']   # 删除键 'Name'
tinydict.clear()       # 清空字典
del tinydict           # 删除字典
```

> **注意**：执行 `del dict` 后，字典不再存在，再次访问会引发 `NameError`。


## 六、字典键的特性

字典值可以是任何的 Python 对象，既可以是标准的对象，也可以是用户定义的，但键不行。

### 6.1 键不允许重复

同一个键出现两次时，**后一个值会覆盖前一个**：

```python
tinydict = {'Name': 'Runoob', 'Age': 7, 'Name': '小菜鸟'}
print(tinydict['Name'])   # 小菜鸟
```

### 6.2 键必须不可变

键可以使用数字、字符串或元组，**但不能使用列表**等可变类型：

```python
# ❌ 错误：列表不能作为键
tinydict = {['Name']: 'Runoob', 'Age': 7}   # TypeError: unhashable type: 'list'
```


## 七、字典内置函数

| 函数 | 描述 | 示例 |
|------|------|------|
| `len(dict)` | 计算字典元素个数（键的总数） | `len(tinydict)` → `3` |
| `str(dict)` | 输出字典的可打印字符串表示 | `str(tinydict)` → `"{'Name': 'Runoob', ...}"` |
| `type(variable)` | 返回变量类型 | `type(tinydict)` → `<class 'dict'>` |


## 八、字典内置方法

| 序号 | 方法 | 描述 |
|------|------|------|
| 1 | `dict.clear()` | 删除字典内所有元素 |
| 2 | `dict.copy()` | 返回字典的浅复制 |
| 3 | `dict.fromkeys(seq, val)` | 创建新字典，以 seq 中元素为键，val 为所有键的初始值 |
| 4 | `dict.get(key, default=None)` | 返回指定键的值，键不存在时返回默认值 |
| 5 | `key in dict` | 检查键是否在字典中，返回 True/False |
| 6 | `dict.items()` | 返回键值对的视图对象 |
| 7 | `dict.keys()` | 返回所有键的视图对象 |
| 8 | `dict.setdefault(key, default=None)` | 与 get() 类似，但键不存在时会添加键并设为默认值 |
| 9 | `dict.update(dict2)` | 将字典 dict2 的键/值对更新到 dict 中 |
| 10 | `dict.values()` | 返回所有值的视图对象 |
| 11 | `dict.pop(key[, default])` | 删除指定键并返回被删除的值 |
| 12 | `dict.popitem()` | 返回并删除字典中的最后一对键和值 |


## 九、字典的常见用法示例

### 9.1 统计元素出现次数

```python
country_counter = {}
def addone(country):
    if country in country_counter:
        country_counter[country] += 1
    else:
        country_counter[country] = 1
```

### 9.2 遍历字典

```python
# 遍历键
for key in dict:
    print(key)

# 遍历值
for value in dict.values():
    print(value)

# 遍历键值对
for key, value in dict.items():
    print(key, value)
```

### 9.3 安全的取值

使用 `get()` 方法可以避免 KeyError：

```python
# ❌ 不安全
value = dict['nonexistent_key']   # KeyError

# ✅ 安全
value = dict.get('nonexistent_key', '默认值')   # 返回 '默认值'
```


## 十、字典 vs 列表

| 对比项 | 字典 | 列表 |
|--------|------|------|
| 索引方式 | 键（任意不可变类型） | 整数索引 |
| 访问速度 | 快速（哈希表） | 较慢（遍历） |
| 有序性 | Python 3.7+ 有序 | 有序 |
| 可变性 | 可变 | 可变 |
