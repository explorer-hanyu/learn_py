# Python3 数据类型转换
## 一、类型转换概述

Python 数据类型转换分为两种：

- **隐式类型转换**：自动完成，不需要用户干预
- **显式类型转换**：需要使用内置函数来完成

## 二、隐式类型转换

### 工作原理
在隐式类型转换中，Python 会自动将一种数据类型转换为另一种数据类型，不需要我们去干预。Python 会将**较低数据类型转换为较高数据类型**（例如整数 → 浮点数），以避免数据丢失。

### 示例：整数 + 浮点数
```python
num_int = 123
num_flo = 1.23
num_new = num_int + num_flo

print("num_int 数据类型为:", type(num_int))   # <class 'int'>
print("num_flo 数据类型为:", type(num_flo))   # <class 'float'>
print("num_new 值为:", num_new)               # 124.23
print("num_new 数据类型为:", type(num_new))   # <class 'float'>
```

> **解析**：Python 自动将整数 `num_int` 转换为浮点数，然后与 `num_flo` 相加，结果存储在浮点数中。

### 不适用隐式转换的情况：整数 + 字符串
```python
num_int = 123
num_str = "456"

print(num_int + num_str)
# TypeError: unsupported operand type(s) for +: 'int' and 'str'
```

Python 无法自动将字符串转换为整数，会抛出 `TypeError`。这种情况需要**显式类型转换**来解决。

## 三、显式类型转换

### 概述
在显式类型转换中，用户将对象的数据类型转换为所需的数据类型。我们使用 `int()`、`float()`、`str()` 等**预定义函数**来执行显式类型转换。

### 3.1 核心转换函数

#### int() — 转换为整数
```python
x = int(1)      # 1
y = int(2.8)    # 2（截断小数，不是四舍五入）
z = int("3")    # 3
```
> **注意**：`int()` 不会四舍五入，而是直接截断小数部分。

#### float() — 转换为浮点数
```python
x = float(1)      # 1.0
y = float(2.8)    # 2.8
z = float("3")    # 3.0
w = float("4.2")  # 4.2
```

#### str() — 转换为字符串
```python
x = str("s1")   # 's1'
y = str(2)      # '2'
z = str(3.0)    # '3.0'
```

#### bool() — 转换为布尔值
```python
# False 的情况
bool(0)      # False
bool(0.0)    # False
bool("")     # False
bool([])     # False
bool({})     # False

# True 的情况
bool(1)      # True
bool("abc")  # True
bool([1,2])  # True
```

> **记忆口诀**：**零、空、None → False**，其他都是 True。

### 3.2 容器类型转换

#### list() — 转换为列表
```python
# 元组 → 列表
my_tuple = (1, 2, 3)
my_list = list(my_tuple)    # [1, 2, 3]

# 字符串 → 列表
my_list = list("abc")       # ['a', 'b', 'c']

# 集合 → 列表
my_list = list({1, 2, 3})   # [1, 2, 3]
```

#### tuple() — 转换为元组
```python
# 列表 → 元组
my_list = [1, 2, 3]
my_tuple = tuple(my_list)   # (1, 2, 3)
```

#### set() — 转换为集合（自动去重）
```python
# 列表 → 集合（去除重复元素）
my_set = set([1, 2, 2, 3])   # {1, 2, 3}

# 字符串 → 集合
my_set = set("hello")        # {'h', 'e', 'l', 'o'}
```

#### dict() — 转换为字典
```python
# 键值对列表 → 字典
my_dict = dict([('name', 'Alice'), ('age', 25)])
# {'name': 'Alice', 'age': 25}
```

### 3.3 进制转换

`int()` 函数还支持一个可选的 **base 参数**，用于将指定进制的字符串转换为十进制整数：

```python
int("FF", 16)    # 255（十六进制 → 十进制）
int("77", 8)     # 63  （八进制 → 十进制）
int("1010", 2)   # 10  （二进制 → 十进制）
```

### 3.4 完整实例：解决整数 + 字符串问题
```python
num_int = 123
num_str = "456"

print("转换前 num_str 类型:", type(num_str))
num_str = int(num_str)        # 显式转换为整数
print("转换后 num_str 类型:", type(num_str))

num_sum = num_int + num_str
print("相加结果:", num_sum)    # 579
print("结果类型:", type(num_sum))
```

## 四、类型转换速查表

### 4.1 基础类型转换函数

| 函数 | 描述 | 示例 | 注意事项 |
|------|------|------|----------|
| `int(x)` | 将 x 转换为整数 | `int(3.14) → 3` | 截断小数，不四舍五入 |
| `float(x)` | 将 x 转换为浮点数 | `float("3") → 3.0` | 字符串必须是合法数字表示 |
| `str(x)` | 将 x 转换为字符串 | `str(3.14) → "3.14"` | 通用性最强 |
| `bool(x)` | 将 x 转换为布尔值 | `bool(0) → False` | 零、空、None 为 False |
| `complex(real, imag)` | 创建复数 | `complex(3, 4) → (3+4j)` | 第二个参数可选 |

### 4.2 容器类型转换函数

| 函数 | 描述 | 示例 | 特点 |
|------|------|------|------|
| `list(x)` | 转换为列表 | `list("abc") → ["a","b","c"]` | 可迭代对象均可转换 |
| `tuple(x)` | 转换为元组 | `tuple([1,2,3]) → (1,2,3)` | 不可变容器 |
| `set(x)` | 转换为集合 | `set([1,2,2,3]) → {1,2,3}` | 自动去重 |
| `dict(x)` | 转换为字典 | `dict([(1,"a")]) → {1:"a"}` | 需要键值对结构 |

## 五、注意事项与最佳实践

| 要点 | 说明 |
|------|------|
| **转换不兼容类型会导致异常** | `int("abc")` → `ValueError` |
| **隐式转换方向** | 较低类型 → 较高类型（避免数据丢失） |
| **直接使用函数名** | 只需将数据类型作为函数名即可使用 |
| **返回新对象** | 转换函数会返回一个新的对象，不会修改原变量 |
| **进制转换** | `int()` 的 `base` 参数支持将其他进制字符串转为十进制 |
