# Python3 基础语法

## 1. 编码
- 默认情况，Python3源码文件以 **UTF-8** 编码，所有字符串都是unicode字符串。
- 可指定不同编码：`# -*- coding: cp-1252 -*-`

## 2. 标识符规则
### 命名规范
- 首字符必须是字母（a-z, A-Z）或下划线 `_`
- 其余部分可由字母、数字、下划线组成
- **大小写敏感**（`count` 和 `Count` 不同）
- 无长度限制，但建议保持简洁
- 禁止使用保留关键字

### 命名风格参考
| 示例 | 含义 |
|------|------|
| `age = 25` | 普通变量名 |
| `user_name = "Alice"` | 下划线连接单词 |
| `_total = 100` | 下划线开头 → "内部使用"/"私有" |
| `MAX_SIZE = 1024` | 全大写 → 常量 |
| `calculate_area()` | 函数名（动词+名词） |
| `StudentInfo` | 类名（首字母大写驼峰） |
| `__private_var` | 双下划线开头（特殊含义） |

### 非法示例
```python
2nd_place = "silver"   # 错误：数字开头
user-name = "Bob"      # 错误：包含连字符
class = "Math"         # 错误：使用关键字opppppppppppppppppp
```

### Python3支持Unicode标识符（可用中文）
```python
姓名 = "张三"   # 合法
π = 3.14159    # 合法
```

### 检查标识符合法性
```python
def is_valid_identifier(name):
    try:
        exec(f"{name} = None")
        return True
    except:
        return False

print(is_valid_identifier("2var"))  # False
print(is_valid_identifier("var2"))  # True
```

## 3. Python保留关键字
> 不能用作标识符名称。可用 `keyword` 模块查看所有关键字：

```python
import keyword
print(keyword.kwlist)
```

|------|--------|------|

| | `None` | 空值 |

## 4. 注释

### 单行注释
以 `#` 开头
```python
# 第一个注释
print("Hello, Python!")  # 第二个注释
```

### 多行注释
可用多个 `#`，或 `'''` / `"""`
```python
# 第一个注释
# 第二个注释

'''
第三注释
第四注释
'''

"""
第五注释
第六注释
"""
```

## 5. 行与缩进
**核心特点**：Python用缩进表示代码块，**不需要大括号 `{}`**。

### 规则
- 缩进空格数可变，但**同一代码块必须使用相同缩进空格数**
- 缩进不一致会导致 `IndentationError`

### 正确示例
```python
if True:
    print("True")
else:
    print("False")
```

### 错误示例（缩进不一致）
```python
if True:
    print("Answer")
    print("True")
else:
    print("Answer")
  print("False")   # 缩进不一致，会报错
```
> 错误信息：`IndentationError: unindent does not match any outer indentation level`

## 6. 多行语句
### 使用反斜杠 `\`
```python
total = item_one + \
        item_two + \
        item_three
```

### 在 `[]`、`{}`、`()` 中不需要反斜杠
```python
total = ['item_one', 'item_two', 'item_three',
         'item_four', 'item_five']
```

## 7. 数字(Number)类型
| 类型 | 说明 | 示例 |
|------|------|------|
| `int` | 整数（只有这一种，无Python2的Long） | `1`, `100` |
| `bool` | 布尔 | `True`, `False` |
| `float` | 浮点数 | `1.23`, `3E-2` |
| `complex` | 复数（a+bj形式） | `1+2j`, `1.1+2.2j` |

## 8. 字符串(String)

### 基本特性
- 单引号 `'` 和双引号 `"` 使用**完全相同**
- 三引号 `'''` 或 `"""` 可指定**多行字符串**
- 反斜杠 `\` 用于转义
- 字符串**不能改变**（不可变）
- Python没有单独的字符类型，字符就是长度为1的字符串

### 常用操作
| 操作 | 示例 | 说明 |
|------|------|------|
| 连接 | `"this " "is " "string"` → `"this is string"` | 字面意义级联 |
| 加法 | `"Hello" + "World"` | 字符串连接 |
| 乘法 | `"Hi" * 3` → `"HiHiHi"` | 字符串重复 |
| 原始字符串 | `r"hello\nrunoob"` | `\n` 不会转义 |
| 索引 | `str[0]`, `str[-1]` | 从左0开始，从右-1开始 |
| 切片 | `str[start:end]` | 包含start，不包含end |
| 切片+步长 | `str[start:end:step]` | step为步长参数 |

### 切片示例
```python
str = '123456789'
print(str)           # 输出字符串
print(str[0:-1])     # 第一个到倒数第二个
print(str[0])        # 第一个字符
print(str[2:5])      # 第3到第6个（不包含第6）
print(str[2:])       # 第3个开始往后所有
print(str[1:5:2])    # 第2到第5，步长2
print(str * 2)       # 输出两次
print(str + '你好')  # 连接字符串

# 转义对比
print('hello\nrunoob')   # 换行输出
print(r'hello\nrunoob')  # 输出 hello\nrunoob
```

## 9. 空行
- 函数之间、类之间、函数入口之间用**空行分隔**，表示新代码的开始
- **空行不是Python语法的一部分**，不插入空行也不会出错
- 作用：分隔不同功能的代码，便于维护和重构
- > 注意：空行也是程序代码的一部分（影响可读性）

## 10. 等待用户输入
```python
#!/usr/bin/python3
input("\n\n按下 enter 键后退出。")
```
> `\n\n` 会在输出前输出两个新空行，按下回车后程序退出。

## 11. 同一行显示多条语句
使用分号 `;` 分割：
```python
import sys; x = 'runoob'; sys.stdout.write(x + '\n')
```

## 12. 代码组
缩进相同的一组语句构成**代码块**（代码组）。复合语句（如 `if`、`while`、`def`、`class`）首行以关键字开始，以冒号 `:` 结束，后续缩进代码构成代码组。
```
if expression :
    suite
elif expression :
    suite
else :
    suite
```

## 13. print输出
默认**换行**，若要**不换行**，在末尾加 `end=""`：
```python
x = "a"
y = "b"

# 换行输出
print(x)
print(y)

# 不换行输出
print(x, end=" ")
print(y, end=" ")
print()
```

## 14. import 与 from...import

| 导入方式 | 语法 |
|----------|------|
| 导入整个模块 | `import somemodule` |
| 导入模块中的特定函数 | `from somemodule import somefunction` |
| 导入模块中的多个函数 | `from somemodule import firstfunc, secondfunc` |
| 导入模块中所有函数 | `from somemodule import *` |

### 示例
```python
# 导入 sys 模块
import sys
print('命令行参数为:')
for i in sys.argv:
    print(i)
print('\n python 路径为', sys.path)

# 导入特定成员
from sys import argv, path
print('path:', path)
```
