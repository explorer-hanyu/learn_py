# Python3 基本数据类型笔记

## 一、变量

### 变量规则
- 变量**不需要声明**，使用前必须**赋值**，赋值后变量才被创建
- 变量本身没有类型，我们所说的"类型"是变量所指的内存中对象的类型
- 等号 `=` 用来给变量赋值，左边是变量名，右边是存储的值

```python
counter = 100          # 整型变量
miles = 1000.0         # 浮点型变量
name = "runoob"        # 字符串变量
```

### 多个变量赋值
```python
# 同时为多个变量赋相同的值
a = b = c = 1

# 同时为多个变量赋不同的值
a, b, c = 1, 2, "runoob"
```

### 查看变量类型
使用 `type()` 函数可以查看变量的类型：

```python
x = 10
y = 3.14
name = "Alice"
is_active = True

print(type(x))      # <class 'int'>
print(type(y))      # <class 'float'>
print(type(name))   # <class 'str'>
print(type(is_active))  # <class 'bool'>
```

### 删除变量
使用 `del` 语句删除对象的引用：
```python
del var
del var_a, var_b, var_c   # 删除多个变量
```

## 二、标准数据类型

### 类型总览
Python3 中有 **6 种标准数据类型**：

| 数据类型 | 中文名称 | 可变性 |
|----------|----------|--------|
| Number | 数字 | 不可变 |
| String | 字符串 | 不可变 |
| bool | 布尔类型 | 不可变 |
| Tuple | 元组 | 不可变 |
| List | 列表 | 可变 |
| Dictionary | 字典 | 可变 |
| Set | 集合 | 可变 |

### 可变/不可变分类
- **不可变数据（4 个）**：Number、String、bool、Tuple
- **可变数据（3 个）**：List、Dictionary、Set

> **补充**：bool 是 int 的子类，有时与其他数字类型合并讨论，但单独列出更为清晰。

## 三、Number（数字）

### 支持的数值类型
Python3 支持 **int、float、bool、complex（复数）**

- **int**：只有一种整数类型 int，表示为长整型，没有 Python2 中的 Long
- **float**：浮点数
- **bool**：布尔类型，是 int 的子类
- **complex**：复数，如 `4+3j`

### 数值运算

| 运算符 | 说明 | 示例 | 结果 |
|--------|------|------|------|
| `+` | 加法 | 5 + 4 | 9 |
| `-` | 减法 | 4.3 - 2 | 2.3 |
| `*` | 乘法 | 3 * 7 | 21 |
| `/` | 除法（返回浮点数） | 2 / 4 | 0.5 |
| `//` | 整除（向下取整） | 2 // 4 | 0 |
| `%` | 取余 | 17 % 3 | 2 |
| `**` | 乘方 | 2 ** 5 | 32 |

**注意**：
- `/` 返回浮点数，`//` 返回整数（向下取整）
- 混合计算时，整型自动转换为浮点数

### 数值类型字面量示例
| int | float | complex |
|-----|-------|---------|
| 10 | 0.0 | 3.14j |
| 100 | 15.20 | 45.j |
| -786 | -21.9 | 9.322e-36j |
| 0o17（八进制） | 32.3e+18 | .876j |
| 0x69（十六进制） | 70.2E-12 | 4.53e-7j |

> **注意**：Python3 禁止整数字面量前导零（如 `080`），八进制必须使用 `0o` 前缀，二进制使用 `0b` 前缀。

### 数值运算示例
```python
print(5 + 4)     # 9
print(4.3 - 2)   # 2.3
print(2 / 4)     # 0.5
print(2 // 4)    # 0
print(17 % 3)    # 2
print(2 ** 5)    # 32
```

### type() 和 isinstance() 的区别

| 函数 | 特点 |
|------|------|
| `type()` | 不会认为子类是父类类型 |
| `isinstance()` | 会认为子类是父类类型 |

```python
class A: pass
class B(A): pass

print(isinstance(A(), A))   # True
print(type(A()) == A)       # True
print(isinstance(B(), A))   # True  (B 是 A 的子类)
print(type(B()) == A)       # False (type 不认为子类是父类)
```

## 四、bool（布尔类型）

### 基础特性
- bool 是 int 的子类，True 等价于 1，False 等价于 0
- 布尔值可以和其他数据类型进行比较、运算

```python
a = True
b = False

print(int(True))    # 1
print(int(False))   # 0

# 布尔运算
print(True and False)   # False
print(True or False)    # True
print(not True)         # False

# 比较运算返回布尔值
print(5 > 3)    # True
print(2 == 2)   # True
```

### 布尔类型转换
使用 `bool()` 函数可将其他类型转换为布尔值。

**以下值转换为布尔值时结果为 `False`：**
- None
- False
- 0（包括 0、0.0、0j）
- 空序列（如 `''`、`()`、`[]`）
- 空映射（如 `{}`）

**其他所有值均为 `True`**

```python
print(bool(0))          # False
print(bool(42))         # True
print(bool(''))         # False
print(bool('Python'))   # True
print(bool([]))         # False
print(bool([1, 2, 3]))  # True
```

### 布尔值在控制流中的应用
```python
x = 10
if x:
    print("x 是非零值，在布尔上下文中为 True")
```

> **提示**：Python 中所有非零数字和非空序列都被视为 True。

## 五、String（字符串）

### 基本特性
- 用单引号 `'` 或双引号 `"` 括起来
- 反斜杠 `\` 转义特殊字符
- 字符串是**不可变**的
- 索引从 0 开始，-1 表示最后一个字符
- `+` 连接字符串，`*` 重复字符串

### 切片语法
```python
变量[头下标:尾下标]   # 包含头下标，不包含尾下标
```

### 字符串操作示例
```python
my_str = 'Runoob'

print(my_str)           # Runoob
print(my_str[0:-1])     # Runoo（第一个到倒数第二个）
print(my_str[0])        # R
print(my_str[2:5])      # noo（索引 2,3,4）
print(my_str[2:])       # noob（索引 2 到末尾）
print(my_str * 2)       # RunoobRunoob
print(my_str + "TEST")  # RunoobTEST
```

### 原始字符串
在字符串前加 `r`，反斜杠不会被转义：
```python
print('Ru\noob')   # Ru（换行）oob
print(r'Ru\noob')  # Ru\noob
```

## 六、List（列表）

### 基本特性
- 写在方括号 `[]` 之间，元素用逗号分隔
- 元素类型可以不同，支持嵌套列表
- **可变类型**：元素可以修改
- `+` 连接列表，`*` 重复列表

### 列表操作示例
```python
my_list = ['abcd', 786, 2.23, 'runoob', 70.2]
tinylist = [123, 'runoob']

print(my_list)          # ['abcd', 786, 2.23, 'runoob', 70.2]
print(my_list[0])       # abcd
print(my_list[1:3])     # [786, 2.23]
print(my_list[2:])      # [2.23, 'runoob', 70.2]
print(tinylist * 2)     # [123, 'runoob', 123, 'runoob']
print(my_list + tinylist)  # ['abcd', 786, 2.23, 'runoob', 70.2, 123, 'runoob']
```

### 列表元素可变
```python
a = [1, 2, 3, 4, 5, 6]
a[0] = 9      # 修改第一个元素
print(a)      # [9, 2, 3, 4, 5, 6]
```

### 列表切片步长
```python
# 步长为 2：每隔一个元素取一个
print(a[1:5:2])   # 索引 1、3

# 步长为负数表示逆向
words = input.split(" ")
words = words[-1::-1]    # 反转列表
```

## 七、Tuple（元组）

### 基本特性
- 写在小括号 `()` 里，元素用逗号隔开
- **不可变类型**：元素不能修改
- 元素类型可以不同
- `+` 连接元组，`*` 重复元组
- 字符串可以被看作一种特殊的元组

### 元组操作示例
```python
my_tuple = ('abcd', 786, 2.23, 'runoob', 70.2)
tinytuple = (123, 'runoob')

print(my_tuple)          # ('abcd', 786, 2.23, 'runoob', 70.2)
print(my_tuple[0])       # abcd
print(my_tuple[1:3])     # (786, 2.23)
print(my_tuple[2:])      # (2.23, 'runoob', 70.2)
print(tinytuple * 2)     # (123, 'runoob', 123, 'runoob')
print(my_tuple + tinytuple)  # ('abcd', 786, 2.23, 'runoob', 70.2, 123, 'runoob')
```

### 元组不可变示例
```python
tup = (1, 2, 3, 4, 5, 6)
tup[0] = 11  # 会报错：TypeError: 'tuple' object does not support item assignment
```

### 特殊语法规则
```python
tup1 = ()       # 空元组
tup2 = (20,)    # 包含一个元素的元组（注意逗号）
not_a_tuple = (42)  # 这是整数 42，不是元组
```

> **注意**：创建单元素元组时，必须在元素后面加逗号，否则会被解释为数学运算括号。

## 八、Set（集合）

### 基本特性
- 使用大括号 `{}` 或 `set()` 函数创建
- **无序**、**可变**的数据类型
- 元素**唯一**（自动去重）
- 支持交集、并集、差集等集合运算

### 创建集合
```python
# 使用大括号
sites = {'Google', 'Taobao', 'Runoob', 'Facebook', 'Zhihu', 'Baidu'}

# 使用 set() 函数（注意：空集合必须用 set()，{} 创建的是空字典）
empty_set = set()
```

### 成员检查
```python
if 'Runoob' in sites:
    print('Runoob 在集合中')
else:
    print('Runoob 不在集合中')
```

### 集合运算
```python
a = set('abracadabra')  # {'a', 'b', 'c', 'd', 'r'}
b = set('alacazam')     # {'a', 'c', 'l', 'm', 'z'}

print(a)              # 集合 a 中的唯一字符
print(a - b)          # 差集：在 a 但不在 b
print(a | b)          # 并集：在 a 或 b
print(a & b)          # 交集：同时在 a 和 b
print(a ^ b)          # 对称差集：在 a 或 b，但不同时存在
```

## 九、Dictionary（字典）

### 基本特性
- 使用大括号 `{}` 标识，是 **键(key): 值(value)** 的集合
- **映射类型**，通过键来访问值
- **键必须使用不可变类型**（如字符串、数字、元组），且不能重复
- **值可以是任意类型**
- Python 3.7 起，字典保持元素的插入顺序

### 字典操作示例
```python
# 创建空字典并添加元素
my_dict = {}
my_dict['one'] = "1 - 菜鸟教程"
my_dict[2] = "2 - 菜鸟工具"

# 创建带初始值的字典
tinydict = {'name': 'runoob', 'code': 1, 'site': 'www.runoob.com'}

print(my_dict['one'])       # 输出键为 'one' 的值
print(my_dict[2])           # 输出键为 2 的值
print(tinydict)             # 输出完整字典
print(tinydict.keys())      # 输出所有键
print(tinydict.values())    # 输出所有值
```

### 使用 dict() 构造函数创建字典
```python
# 从键值对序列构建
dict([('Runoob', 1), ('Google', 2), ('Taobao', 3)])
# {'Runoob': 1, 'Google': 2, 'Taobao': 3}

# 字典推导式
{x: x**2 for x in (2, 4, 6)}
# {2: 4, 4: 16, 6: 36}

# 关键字参数方式
dict(Runoob=1, Google=2, Taobao=3)
# {'Runoob': 1, 'Google': 2, 'Taobao': 3}
```

### 字典要点总结
- 字典是映射类型，元素是键值对
- 键必须是不可变类型，且不能重复
- 创建空字典使用 `{}`，创建空集合使用 `set()`
- Python 3.7 起字典保持插入顺序

## 十、bytes 类型

### 基本特性
- 表示**不可变的二进制序列**
- 元素是整数值（0 到 255），不是 Unicode 字符
- 常用于处理图像、音频、视频等二进制数据
- 在网络编程中用于传输二进制数据

### 创建 bytes 对象
```python
# 使用 b 前缀
x = b"hello"
print(x)            # b'hello'
print(type(x))      # <class 'bytes'>
print(x[0])         # 104（'h' 的 ASCII 值）

# 使用 bytes() 函数
x = bytes("hello", encoding="utf-8")
```

### bytes 操作
```python
x = b"hello"
y = x[1:3]        # 切片：b'el'
z = x + b"world"  # 拼接：b'helloworld'

print(y)  # b'el'
print(z)  # b'helloworld'
```

### 字符与整数比较
```python
x = b"hello"
if x[0] == ord("h"):   # ord("h") 返回 104
    print("第一个元素是 'h'")
```

> **注意**：bytes 类型是不可变的，修改操作需要创建新的 bytes 对象。

## 十一、数据类型转换（预告）

以下内置函数可执行数据类型转换，返回新对象：

| 函数 | 描述 |
|------|------|
| `int(x)` | 将 x 转换为整数 |
| `float(x)` | 将 x 转换为浮点数 |
| `complex(real, imag)` | 创建复数 |
| `str(x)` | 将对象 x 转换为字符串 |
| `repr(x)` | 将对象 x 转换为表达式字符串 |
| `eval(str)` | 计算字符串中的有效表达式并返回对象 |
| `tuple(s)` | 将序列 s 转换为元组 |
| `list(s)` | 将序列 s 转换为列表 |
| `set(s)` | 转换为可变集合 |
| `dict(d)` | 创建字典 |
| `frozenset(s)` | 转换为不可变集合 |
| `chr(x)` | 将整数转换为对应字符 |
| `ord(x)` | 将字符转换为整数值 |
| `hex(x)` | 将整数转换为十六进制字符串 |
| `oct(x)` | 将整数转换为八进制字符串 |
