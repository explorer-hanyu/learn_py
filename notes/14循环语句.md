# Python3 循环语句笔记

## 一、循环概述

Python 中的循环语句有 **`for`** 和 **`while`** 两种。

### 循环控制关键字速查

| 关键字/函数 | 说明 | 示例 |
|------------|------|------|
| `for` | 迭代循环，用于遍历序列或可迭代对象 | `for i in list:` |
| `while` | 条件循环，条件为 True 时持续执行 | `while x > 0:` |
| `break` | 立即终止当前循环 | `break` |
| `continue` | 跳过本次循环剩余代码，进入下一次迭代 | `continue` |
| `else`（循环） | 循环正常结束（未被 `break`）时执行 | `for i in range(3): ... else: ...` |
| `pass` | 循环中的占位语句（空操作） | `for i in range(5): pass` |
| `range()` | 生成整数序列，常与 `for` 循环配合使用 | `range(0, 5)` |
| `enumerate()` | 遍历时同时获取索引和值 | `for i, v in enumerate(list):` |


## 二、while 循环

### 2.1 基本语法

Python 中 `while` 语句的一般形式：

```python
while 判断条件(condition):
    执行语句(statements)……
```

**关键点**：
- 同样需要注意**冒号**和**缩进**
- Python 中**没有 `do..while` 循环**

### 2.2 示例：计算 1 到 100 的总和

```python
#!/usr/bin/env python3
n = 100
sum = 0
counter = 1
while counter <= n:
    sum = sum + counter
    counter += 1
print("1 到 %d 之和为: %d" % (n, sum))
# 输出：1 到 100 之和为: 5050
```


### 2.3 无限循环

通过设置条件表达式**永远不为 False** 来实现无限循环：

```python
#!/usr/bin/python3
var = 1
while var == 1:   # 表达式永远为 true
    num = int(input("输入一个数字 :"))
    print("你输入的数字是: ", num)
print("Good bye!")
```


> **提示**：可以使用 `CTRL+C` 来退出无限循环。无限循环在服务器上处理客户端的实时请求时非常有用。

### 2.4 while 循环使用 else

如果 `while` 后面的条件语句为 `False` 时，则执行 `else` 的语句块：

```python
#!/usr/bin/python3
count = 0
while count < 5:
    print(count, " 小于 5")
    count = count + 1
else:
    print(count, " 大于或等于 5")
```


**输出**：
```
0 小于 5
1 小于 5
2 小于 5
3 小于 5
4 小于 5
5 大于或等于 5
```


### 2.5 简单语句组

如果 `while` 循环体中只有一条语句，可以将该语句与 `while` 写在同一行：

```python
#!/usr/bin/python
flag = 1
while (flag): print('欢迎访问菜鸟教程!')
print("Good bye!")
```



## 三、for 语句

### 3.1 基本语法

Python `for` 循环可以遍历任何可迭代对象，如一个列表或者一个字符串：

```python
for <变量> in <序列>:
    <执行语句>
else:
    <循环正常结束执行的语句>
```


### 3.2 遍历列表

```python
#!/usr/bin/python3
sites = ["Baidu", "Google", "Runoob", "Taobao"]
for site in sites:
    print(site)
```


### 3.3 遍历字符串

```python
#!/usr/bin/python3
word = 'runoob'
for letter in word:
    print(letter)
```


**输出**：
```
r
u
n
o
o
b
```


### 3.4 配合 range() 使用

```python
#!/usr/bin/python3
# 1 到 5 的所有数字：
for number in range(1, 6):
    print(number)
```


**输出**：
```
1
2
3
4
5
```



## 四、循环控制语句

### 4.1 break 语句

`break` 用于**立即终止当前循环**，跳出循环体。

```python
#!/usr/bin/python3
sites = ["Baidu", "Google", "Runoob", "Taobao"]
for site in sites:
    if site == "Runoob":
        print("菜鸟教程!")
        break
    print("循环数据 " + site)
else:
    print("没有循环数据!")
print("完成循环!")
```


**输出**：
```
循环数据 Baidu
循环数据 Google
菜鸟教程!
完成循环!
```


### 4.2 continue 语句

`continue` 用于**跳过本次循环剩余代码**，进入下一次迭代。

### 4.3 pass 语句

`pass` 是**空语句**，用于保持程序结构的完整性，在语法上需要一条语句但程序不需要任何操作时使用。

```python
for i in range(5):
    pass   # 占位，什么都不做
```


## 五、range() 函数

### 5.1 基本用法

如果你需要遍历数字序列，可以使用内置 `range()` 函数：

```python
>>> for i in range(5):
...     print(i)
...
0
1
2
3
4
```


### 5.2 指定区间

```python
>>> for i in range(5, 9):
...     print(i)
...
5
6
7
8
```


### 5.3 指定步长

可以指定数字开始并指定不同的增量（步长），甚至可以是负数：

```python
>>> for i in range(0, 10, 3):
...     print(i)
...
0
3
6
9
```


```python
>>> for i in range(-10, -100, -30):
...     print(i)
...
-10
-40
-70
```


### 5.4 结合 len() 遍历索引

可以结合 `range()` 和 `len()` 函数来遍历一个序列的索引：

```python
>>> a = ['Google', 'Baidu', 'Runoob', 'Taobao', 'QQ']
>>> for i in range(len(a)):
...     print(i, a[i])
...
0 Google
1 Baidu
2 Runoob
3 Taobao
4 QQ
```


### 5.5 创建列表

`range()` 函数还可以用来创建一个列表：

```python
>>> list(range(5))
[0, 1, 2, 3, 4]
```



## 六、for...else 语句

在 Python 中，`for...else` 语句用于在循环结束后执行一段代码。

**语法格式**：

```python
for item in iterable:
    # 循环主体
else:
    # 循环结束后执行的代码
```


**核心规则**：
- 当循环执行完毕（即遍历完 iterable 中的所有元素）后，会执行 `else` 子句中的代码
- 如果在循环过程中遇到了 `break` 语句，则会中断循环，**此时不会执行 `else` 子句**

```python
for x in range(6):
    print(x)
else:
    print("Finally finished!")
```


**输出**：
```
0
1
2
3
4
5
Finally finished!
```



## 七、嵌套循环

Python 允许在一个循环体内嵌入另一个循环。可以在 `while` 循环中嵌套 `for` 循环，也可以在 `for` 循环中嵌套 `while` 循环。

**语法示例**：

```python
# for 循环嵌套 for 循环
for i in range(3):
    for j in range(2):
        print(f"i={i}, j={j}")

# while 循环嵌套 while 循环
i = 0
while i < 3:
    j = 0
    while j < 2:
        print(f"i={i}, j={j}")
        j += 1
    i += 1
```


## 八、重点总结

| 知识点 | 要点 |
|--------|------|
| **while 循环** | 条件为 True 时持续执行，注意不要写成死循环 |
| **for 循环** | 遍历可迭代对象（列表、字符串、range 等） |
| **break** | 立即终止整个循环 |
| **continue** | 跳过本次循环剩余代码，进入下一次迭代 |
| **else（循环）** | 循环正常结束（未被 break 中断）时执行 |
| **pass** | 空操作占位符，保持语法完整 |
| **range()** | 生成整数序列，支持 start、stop、step 三个参数 |
| **嵌套循环** | 循环内部可以嵌套另一个循环 |
