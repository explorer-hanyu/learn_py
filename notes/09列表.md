# Python3 列表(List)
## 一、序列与列表概述

**序列**是 Python 中最基本的数据结构。序列中的每个值都有对应的位置值，称之为**索引**，第一个索引是 0，第二个索引是 1，依此类推。

Python 有 6 个序列的内置类型，但最常见的是**列表**和**元组**。

**列表**是最常用的 Python 数据类型，它可以作为一个方括号内的逗号分隔值出现。

### 列表的核心特性

| 特性 | 说明 |
|------|------|
| **可变性** | 列表是**可变的**，可以修改、添加、删除元素 |
| **元素类型** | 列表的数据项**不需要具有相同的类型** |
| **有序性** | 列表中的元素按插入顺序排列，每个元素有对应的索引 |
| **嵌套支持** | 列表可以包含其他列表（嵌套列表） |


## 二、创建列表

创建一个列表，只要把逗号分隔的不同的数据项使用**方括号 `[]`** 括起来即可。

```python
list1 = ['Google', 'Runoob', 1997, 2000]   # 混合类型
list2 = [1, 2, 3, 4, 5]                    # 纯数字
list3 = ["a", "b", "c", "d"]               # 纯字符串
list4 = ['red', 'green', 'blue', 'yellow', 'white', 'black']
```


## 三、访问列表中的值

### 3.1 正向索引

与字符串的索引一样，列表索引从 **0** 开始：

```python
list = ['red', 'green', 'blue', 'yellow', 'white', 'black']
print(list[0])   # red
print(list[1])   # green
print(list[2])   # blue
```

### 3.2 负向索引

索引也可以从尾部开始，最后一个元素的索引为 **-1**，往前一位为 -2，以此类推：

```python
list = ['red', 'green', 'blue', 'yellow', 'white', 'black']
print(list[-1])   # black
print(list[-2])   # white
print(list[-3])   # yellow
```

### 3.3 切片操作

使用方括号 `[]` 形式截取列表：

```python
nums = [10, 20, 30, 40, 50, 60, 70, 80, 90]
print(nums[0:4])   # [10, 20, 30, 40]
```

使用负数索引值截取：

```python
list = ['Google', 'Runoob', 'Zhihu', 'Taobao', 'Wiki']
print(list[1])        # Runoob
print(list[1:-2])     # ['Runoob', 'Zhihu']（从索引1到倒数第2个，不包含倒数第2个）
```


## 四、更新列表

你可以对列表的数据项进行修改或更新，也可以使用 `append()` 方法来添加列表项：

```python
list = ['Google', 'Runoob', 1997, 2000]
print("第三个元素为 : ", list[2])   # 1997

# 修改元素
list[2] = 2001
print("更新后的第三个元素为 : ", list[2])   # 2001

# 追加元素
list1 = ['Google', 'Runoob', 'Taobao']
list1.append('Baidu')
print("更新后的列表 : ", list1)   # ['Google', 'Runoob', 'Taobao', 'Baidu']
```


## 五、删除列表元素

可以使用 `del` 语句来删除列表的的元素：

```python
list = ['Google', 'Runoob', 1997, 2000]
print("原始列表 : ", list)
del list[2]
print("删除第三个元素后 : ", list)   # ['Google', 'Runoob', 2000]
```


## 六、列表脚本操作符

列表对 `+` 和 `*` 的操作符与字符串相似：

| 操作符 | 描述 | 示例 |
|--------|------|------|
| `+` | 组合列表 | `[1,2] + [3,4]` → `[1,2,3,4]` |
| `*` | 重复列表 | `[1,2] * 2` → `[1,2,1,2]` |
| `in` | 检查成员 | `3 in [1,2,3]` → `True` |
| `not in` | 检查非成员 | `4 not in [1,2,3]` → `True` |


## 七、列表常用方法

| 方法 | 描述 | 示例 |
|------|------|------|
| `append(obj)` | 在列表末尾添加新的对象 | `list.append('Baidu')` |
| `extend(seq)` | 在列表末尾一次性追加另一个序列中的多个值 | `list.extend([1,2,3])` |
| `insert(index, obj)` | 在指定位置插入对象 | `list.insert(1, 'A')` |
| `remove(obj)` | 移除列表中第一个匹配项 | `list.remove('Runoob')` |
| `pop([index])` | 移除指定位置的元素，默认移除最后一个 | `list.pop()` |
| `clear()` | 清空列表 | `list.clear()` |
| `index(obj)` | 返回第一个匹配项的索引 | `list.index('Runoob')` |
| `count(obj)` | 统计某个元素在列表中出现的次数 | `list.count(2)` |
| `sort()` | 对列表进行排序 | `list.sort()` |
| `reverse()` | 反向列表中的元素 | `list.reverse()` |
| `copy()` | 返回列表的浅拷贝 | `list.copy()` |

### 方法详解

#### extend() — 扩展列表

`extend()` 函数用于在列表末尾一次性追加另一个序列中的多个值。

参数可以是列表、元组、集合、字典；若为字典，则仅会将**键(key)** 作为元素依次添加至原列表的末尾。

```python
list1 = ['Google', 'Runoob', 'Taobao']
list2 = list(range(5))   # 创建 0-4 的列表
list1.extend(list2)
print("扩展后的列表：", list1)   # ['Google', 'Runoob', 'Taobao', 0, 1, 2, 3, 4]
```

不同数据类型的扩展：

```python
language = ['French', 'English', 'German']
language_tuple = ('Spanish', 'Portuguese')
language_set = {'Chinese', 'Japanese'}

language.extend(language_tuple)   # 添加元组
language.extend(language_set)     # 添加集合
print(language)   # ['French', 'English', 'German', 'Spanish', 'Portuguese', 'Chinese', 'Japanese']
```

#### reverse() — 反向列表

`reverse()` 函数用于反向列表中元素，**没有返回值**，但会对列表的元素进行反向排序。

```python
list1 = ['Google', 'Runoob', 'Taobao', 'Baidu']
list1.reverse()
print("列表反转后: ", list1)   # ['Baidu', 'Taobao', 'Runoob', 'Google']
```

#### clear() — 清空列表

`clear()` 函数用于清空列表，类似于 `del a[:]`。

```python
RUNOOB = [6, 0, 4, 1]
print('清空前:', RUNOOB)   # [6, 0, 4, 1]
RUNOOB.clear()
print('清空后:', RUNOOB)   # []
```


## 八、列表常用函数

| 函数 | 描述 | 示例 |
|------|------|------|
| `len(list)` | 返回列表元素个数 | `len([1,2,3])` → `3` |
| `max(list)` | 返回列表元素中的最大值 | `max([1,5,3])` → `5` |
| `min(list)` | 返回列表元素中的最小值 | `min([1,5,3])` → `1` |
| `list(seq)` | 将元组或字符串转换为列表 | `list("abc")` → `['a','b','c']` |

### max() 的比较规则

- **全部为数字类型**：直接根据值的大小比较
- **全部为字符串类型**：比较每个字符串元素的**第一个字符的 ASCII 码**大小
- **数字与字符串混杂**：无法比较

```python
list1 = ['Google', 'Runoob', 'Taobao']
list2 = [456, 700, 200]
print(max(list1))   # Taobao
print(max(list2))   # 700
```

### list() — 转换为列表

`list()` 方法用于将**元组或字符串**转换为列表。

```python
aTuple = (123, 'Google', 'Runoob', 'Taobao')
list1 = list(aTuple)
print("列表元素 : ", list1)   # [123, 'Google', 'Runoob', 'Taobao']

str = "Hello World"
list2 = list(str)
print("列表元素 : ", list2)   # ['H', 'e', 'l', 'l', 'o', ' ', 'W', 'o', 'r', 'l', 'd']
```
