# Python3 集合(Set)


## 一、集合概述

集合（set）是一个**无序的、不重复的元素序列**。

### 集合的核心特性

| 特性 | 说明 |
|------|------|
| **无序性** | 集合中的元素没有顺序，不能通过索引访问 |
| **唯一性** | 集合中的元素**不会重复**，自动去重 |
| **可变性** | 集合是**可变的**，可以添加或删除元素 |
| **元素要求** | 元素必须是**不可变类型**（数字、字符串、元组等），列表等可变类型不能作为集合元素 |


## 二、创建集合

### 2.1 使用花括号 `{}` 创建

```python
set1 = {1, 2, 3, 4}              # 直接使用大括号创建集合
set2 = {'apple', 'orange', 'banana'}
```

### 2.2 使用 `set()` 函数创建

```python
set3 = set([4, 5, 6, 7])         # 从列表创建集合
set4 = set('abracadabra')        # 从字符串创建集合，自动去重
# 结果: {'a', 'r', 'b', 'c', 'd'}
```

### ⚠️ 创建空集合

**创建一个空集合必须用 `set()` 而不是 `{}`**，因为 `{}` 是用来创建一个空字典的。

```python
empty_set = set()     # ✅ 空集合
empty_dict = {}       # ❌ 这是空字典，不是空集合
```


## 三、集合的基本操作

### 3.1 添加元素 — `add()`

将元素 x 添加到集合中，**如果元素已存在，则不进行任何操作**。

```python
thisset = set(("Google", "Runoob", "Taobao"))
thisset.add("Facebook")
print(thisset)   # {'Taobao', 'Facebook', 'Google', 'Runoob'}
```

### 3.2 添加多个元素 — `update()`

参数可以是列表、元组、字典等，**可以有多个参数，用逗号分开**。

```python
thisset = set(("Google", "Runoob", "Taobao"))
thisset.update({1, 3})              # 添加集合
print(thisset)   # {1, 3, 'Google', 'Taobao', 'Runoob'}

thisset.update([1, 4], [5, 6])      # 添加多个列表
print(thisset)   # {1, 3, 4, 5, 6, 'Google', 'Taobao', 'Runoob'}
```

### 3.3 移除元素 — `remove()`

将元素 x 从集合中移除，**如果元素不存在，则会发生 KeyError 错误**。

```python
thisset = set(("Google", "Runoob", "Taobao"))
thisset.remove("Taobao")
print(thisset)   # {'Google', 'Runoob'}

thisset.remove("Facebook")   # 不存在 → KeyError
```

### 3.4 安全移除元素 — `discard()`

移除集合中的元素，**如果元素不存在，不会发生错误**。

```python
thisset = set(("Google", "Runoob", "Taobao"))
thisset.discard("Facebook")   # 不存在也不会报错
print(thisset)   # {'Taobao', 'Google', 'Runoob'}
```

### 3.5 随机移除元素 — `pop()`

**随机删除**集合中的一个元素并返回该元素。

```python
thisset = set(("Google", "Runoob", "Taobao", "Facebook"))
x = thisset.pop()
print(x)   # 每次执行结果可能不同
```

> **注意**：`pop()` 方法会对集合进行无序排列，然后删除左面第一个元素。

### 3.6 计算集合元素个数 — `len()`

```python
thisset = set(("Google", "Runoob", "Taobao"))
print(len(thisset))   # 3
```

### 3.7 清空集合 — `clear()`

```python
thisset = set(("Google", "Runoob", "Taobao"))
thisset.clear()
print(thisset)   # set()
```

### 3.8 判断元素是否存在 — `in`

```python
thisset = set(("Google", "Runoob", "Taobao"))
print("Runoob" in thisset)     # True
print("Facebook" in thisset)   # False
```


## 四、集合运算

### 4.1 差集 — `-`

集合 a 中包含而集合 b 中**不包含**的元素。

```python
a = set('abracadabra')   # {'a', 'r', 'b', 'c', 'd'}
b = set('alacazam')      # {'a', 'c', 'l', 'm', 'z'}
print(a - b)   # {'r', 'd', 'b'}
```

### 4.2 并集 — `|`

集合 a 或 b 中**包含的所有元素**。

```python
print(a | b)   # {'a', 'c', 'r', 'd', 'b', 'm', 'z', 'l'}
```

### 4.3 交集 — `&`

集合 a 和 b 中**都包含**的元素。

```python
print(a & b)   # {'a', 'c'}
```

### 4.4 对称差集 — `^`

**不同时包含**于 a 和 b 的元素（即在 a 或在 b，但不同时在两者中）。

```python
print(a ^ b)   # {'r', 'd', 'b', 'm', 'z', 'l'}
```


## 五、集合推导式

类似列表推导式，集合也支持**集合推导式（Set Comprehension）**。

```python
a = {x for x in 'abracadabra' if x not in 'abc'}
print(a)   # {'r', 'd'}
```


## 六、集合内置方法速查表

| 方法 | 描述 |
|------|------|
| `add()` | 为集合添加元素 |
| `clear()` | 移除集合中的所有元素 |
| `copy()` | 拷贝一个集合 |
| `difference()` | 返回多个集合的差集 |
| `difference_update()` | 移除集合中的元素，该元素在指定的集合中也存在 |
| `discard()` | 删除集合中指定的元素 |
| `intersection()` | 返回集合的交集 |
| `intersection_update()` | 返回集合的交集 |
| `isdisjoint()` | 判断两个集合是否包含相同的元素，如果没有返回 True |
| `issubset()` | 判断指定集合是否为该方法参数集合的子集 |
| `issuperset()` | 判断该方法的参数集合是否为指定集合的子集 |
| `pop()` | 随机移除元素 |
| `remove()` | 移除指定元素 |
| `symmetric_difference()` | 返回两个集合中不重复的元素集合 |
| `symmetric_difference_update()` | 移除当前集合中在另一个指定集合中相同的元素，并将另一个集合中不同的元素插入到当前集合中 |
| `union()` | 返回两个集合的并集 |
| `update()` | 给集合添加元素 |
| `len()` | 计算集合元素个数 |


## 七、集合 vs 列表 vs 元组 vs 字典

| 对比项 | 集合 | 列表 | 元组 | 字典 |
|--------|------|------|------|------|
| 符号 | `{}` / `set()` | `[]` | `()` | `{}` |
| 有序性 | **无序** | 有序 | 有序 | Python 3.7+ 有序 |
| 可变性 | 可变 | 可变 | **不可变** | 可变 |
| 元素唯一性 | **唯一** | 可重复 | 可重复 | 键唯一 |
| 索引方式 | 不支持索引 | 整数索引 | 整数索引 | 键 |
| 空对象 | `set()` | `[]` | `()` | `{}` |

## 任务：电影推荐系统（集合专项练习）

编写一个名为 `movie_recommend.py` 的程序，综合运用集合的各种操作完成以下功能。

### 要求

1. **创建集合**
   - 使用**花括号**创建集合 `user1_movies`，包含电影：`"盗梦空间"`, `"星际穿越"`, `"泰坦尼克号"`, `"阿凡达"`
   - 使用 `set()` 函数从列表创建集合 `user2_movies`，包含电影：`"星际穿越"`, `"阿凡达"`, `"复仇者联盟"`, `"盗梦空间"`
   - 创建一个**空集合** `common_movies`（注意：不能使用 `{}`）

2. **集合基本操作**
   - 使用 `add()` 为 `user1_movies` 添加一部新电影 `"流浪地球"`
   - 使用 `update()` 为 `user2_movies` 同时添加 `"哪吒"` 和 `"姜子牙"`（使用列表作为参数）
   - 使用 `len()` 分别打印两个集合的元素个数
   - 使用 `in` 检查 `"泰坦尼克号"` 是否在 `user1_movies` 中

3. **集合运算**
   - 计算并打印**交集**：两位用户共同看过的电影
   - 计算并打印**并集**：两位用户看过的所有电影（去重）
   - 计算并打印**差集**：`user1_movies - user2_movies`（user1 看过但 user2 没看过的）
   - 计算并打印**对称差集**：只有一位用户看过的电影

4. **添加与移除元素**
   - 使用 `remove()` 从 `user1_movies` 中移除 `"泰坦尼克号"`
   - 使用 `discard()` 从 `user2_movies` 中移除 `"哈利波特"`（该电影不存在，观察是否报错）
   - 使用 `pop()` 随机从 `user1_movies` 中移除一部电影，并打印被移除的电影名

5. **集合推导式**
   - 给定列表 `movies = ["盗梦空间", "星际穿越", "盗梦空间", "阿凡达", "星际穿越", "流浪地球"]`
   - 使用**集合推导式**创建一个新集合，包含列表中所有**长度大于 3** 的电影名（自动去重）

6. **综合挑战：电影推荐**
   - 编写代码，根据两位用户的观影记录，推荐 `user2` 没看过但 `user1` 看过的电影（即差集 `user1_movies - user2_movies`）
   - 如果推荐列表不为空，打印 `"推荐你看：xxx"`；否则打印 `"没有新电影推荐"`

7. **内置方法练习**
   - 使用 `copy()` 创建 `user1_movies` 的副本 `user1_copy`
   - 向 `user1_copy` 中添加一部新电影 `"战狼2"`
   - 打印原集合和副本，观察是否相互影响（用注释说明结果）
   - 使用 `clear()` 清空 `user1_copy`，打印确认
